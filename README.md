# 阿里云OSS
本服务用于管理阿里云OSS,本服务是对阿里云SDK的封装，所以方法名与阿里的SDK一样，下面是列举出的几个使用方法，其他OSS方法都是可以使用的，请参考阿里云 [OSS参考文档](https://help.aliyun.com/document_detail/32103.html?spm=5176.doc32099.6.748.85Qz6b)。
[TOC]

#### 安装组件
使用 composer 命令进行安装或下载源代码使用。

```
composer require houdunwang/oss
```

## 组件配置

```
Config::set('oss',[
                      /*
                      |--------------------------------------------------------------------------
                      | 登录阿里云访问控制查看
                      |--------------------------------------------------------------------------
                      | https://ram.console.aliyun.com/
                      */
                      'accessKeyId'     => '',
                      'accessKeySecret' => '',
                  
                      /*
                      |--------------------------------------------------------------------------
                      | Endpoint
                      |--------------------------------------------------------------------------
                      | 登录阿里云后台查看,可以设置阿里云提供的公共域名，也可以使用自定义域名。
                      | 如果使用自定义域名，需要将下面的 "自定义域名" 设置为 true
                      */
                      'endpoint'        => 'http://oss-cn-hangzhou.aliyuncs.com',
                  
                      /*
                      |--------------------------------------------------------------------------
                      | 自定义域名
                      |--------------------------------------------------------------------------
                      | 是否使用自定义域名，需要先在阿里云后台OSS业务处做好域名解析
                      */
                      'custom_domain'   => false,
                  
                      /*
                      |--------------------------------------------------------------------------
                      | 外网域名
                      |--------------------------------------------------------------------------
                      | 外网域名比如会在前端上传获取上传签名时使用
                      */
                      'host'        => 'http://hdphp.oss-cn-hangzhou.aliyuncs.com',
                  
                      /*
                      |--------------------------------------------------------------------------
                      | Bucket块名称
                      |--------------------------------------------------------------------------
                      | https://oss.console.aliyun.com/index
                      */
                      'bucket'          => 'hdcms',
                  ]
);
```

## 字符串上传
```
$object = "hd.txt";
$content = "Hi, OSS.";
Oss::putObject($object, $content);
```

## 上传本地文件
```
$object = "1.mp4";
$filePath = '/www/1.mp4';
Oss::uploadFile($object, $filePath);
```

## 生成前端上传签名
```
Oss::sign();
```
